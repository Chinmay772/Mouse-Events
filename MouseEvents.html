Understanding Javascript Mouse Events

MouseOver event triggered:

0

Triggered when the mouse pointer enters the
area of an element or its child elements.

MouseEnter event triggered:

0

Triggered when the mouse pointer enters the
area of an element.

Mouse Click event triggered:

0

Triggered when the mouse button is clicked
(pressed and released) on an element.

Mousemove event triggered:

0

Triggered when the mouse pointer is moved while
it is over an element.

MouseDown event triggered:

0

Triggered when the mouse button is pressed down
over an element.

MouseUp event triggered:

0

Triggered when the mouse button is released after
being pressed down.

MouseOut event triggered:

0

Triggered when the mouse pointer leaves the
area of an element or its child elements.

Wheel event triggered:

0

Triggered when the mouse wheel is rotated.

Double Click event triggered:

0

Triggered when the element is double-clicked
with the mouse.
